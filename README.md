# 📞 Análisis de Eficiencia Operativa - CallMeMaybe
### Optimización de Desempeño y Recuperación de Clientes en Telecomunicaciones

---

## 📝 Contexto del Proyecto
En este proyecto, analicé el desempeño de **1,092 operadores** de un servicio de telefonía virtual. El objetivo principal fue identificar ineficiencias en la atención telefónica que estaban resultando en llamadas perdidas y tiempos de espera prolongados, afectando directamente la retención de clientes.

## 🎯 Objetivos Clave
* **Detección de Ineficiencias:** Identificar operadores con bajo desempeño mediante el análisis de tasas de llamadas perdidas.
* **Optimización de Tiempos:** Evaluar y reducir los tiempos de espera (Wait Times) por cliente.
* **Validación Estadística:** Asegurar que los hallazgos no sean aleatorios mediante pruebas de hipótesis.

## 🛠️ Stack Tecnológico
| Herramienta | Uso Principal |
| :--- | :--- |
| **Python (Pandas/NumPy)** | Limpieza de datos, procesamiento de datasets masivos y ETL. |
| **Matplotlib / Seaborn** | Visualización de tendencias y distribución de llamadas por operador. |
| **SciPy (Estadística)** | Aplicación de Pruebas T de Student para validación de hipótesis. |

## 📊 Resultados e Impacto de Negocio
Gracias al análisis de datos, se lograron identificar los siguientes puntos accionables:
* ✅ **Productividad:** Incremento proyectado del **15%** en la eficiencia general del Call Center.
* 📞 **Recuperación:** Identificación de **1,086 llamadas perdidas** que pueden ser recuperadas mediante una mejor asignación de turnos.
* ⏱️ **Experiencia del Cliente:** Reducción detectada de **2.2 minutos** en el tiempo promedio de espera por cliente.
* 🔬 **Rigor Científico:** Validación de resultados con un **p-value < 0.05**, confirmando que las mejoras propuestas son estadísticamente significativas.
